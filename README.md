# 🎬 Movie Review Sentiment Analysis Using Sentence Embeddings

A LLM application that performs sentiment analysis on movie reviews using **Sentence Transformers embeddings** and **Logistic Regression**. The project uses the **IMDb Movie Reviews Dataset** and provides both a command-line interface and an interactive Streamlit web application.

---

## 📌 Project Overview

This project demonstrates how modern sentence embeddings can be combined with a traditional machine learning classifier to build an accurate sentiment analysis system.

Instead of using Bag-of-Words or TF-IDF features, reviews are converted into dense semantic embeddings using the **all-MiniLM-L6-v2** Sentence Transformer model. These embeddings are then used to train a Logistic Regression classifier for binary sentiment classification.

The model predicts whether a movie review is:

* ✅ Positive
* ❌ Negative

## 🚀 Features

## 📚 Dataset

This project uses the IMDb Movie Reviews Dataset from Hugging Face:

* Dataset: `stanfordnlp/imdb`
* Total Reviews: 50,000
* Classes:

  * Positive (1)
  * Negative (0)

For faster experimentation:

* Training Samples Used: 5,000
* Test Samples Used: 1,000

---

## 🧠 Model Architecture

### Step 1: Text Embedding

Reviews are converted into dense vector representations using:

```python
SentenceTransformer("all-MiniLM-L6-v2")
```

The model generates semantic embeddings that capture the meaning of the review text.

### Step 2: Classification

The embeddings are passed to a Logistic Regression classifier:

```python
LogisticRegression()
```

The classifier learns patterns associated with positive and negative sentiments.
